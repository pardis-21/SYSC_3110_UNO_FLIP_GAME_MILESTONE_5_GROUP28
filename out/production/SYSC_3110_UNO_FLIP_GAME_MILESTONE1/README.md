Project Overview Milestone 1:
This prroject is a simplified UNO-style card game implemented in Java.
It includes the main gameplay logic, player management and card functionalities,
all supported by unit tests and UML/Sequence diagrams to illustrate the system.

The main deliverables include:
Complete implementation of all classes(Card, Player, PlayOrder, GameLogic and Game)
UML and sequence diagrams representing the system's structure and interactions.
unit tests for all classes and methods.
